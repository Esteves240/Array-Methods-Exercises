forEach        → percorre o array (não retorna nada)
map            → transforma cada elemento (retorna novo array)
filter         → filtra elementos com base numa condição (retorna novo array)

reduce         → reduz o array a um único valor
    array.reduce((acumulador, valorAtual) => {
        return novoAcumulador;
    }, valorInicial);

Spread (...)   → espalha valores (arrays/objetos)
Rest (...)     → agrupa valores restantes
Destructuring  → extrai valores de arrays ou objetos
